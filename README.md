# Data Science Internship Project

This repository contains my Data Science internship work involving Python, EDA, Machine Learning, Deep Learning, YOLO, and NLP.

##  Tech Stack
Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, TensorFlow/PyTorch, YOLO,OpenCV, NLP (NLTK/spaCy),Streamlit.

## Key Features
- Data preprocessing and EDA  
- Machine Learning model building( Linear Regression, Logistic Regression)
- Deep Learning implementations
- Model deployment using Streamlit
- YOLO object detection  
- NLP tasks like tokenization, sentiment analysis.
